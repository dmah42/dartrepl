dartrepl
========

A REPL for the dart language.  Uses the same dart binary that is running 
this app.

Works with build 17072.

Example usage:
    `c:\dart.exe dartrepl.dart`
    
    >> main() => print("Hello World");
      << Hello World
    [success]
    >> // todo, enter another command...!
